# Convertidor MP4 a MP3

Pequeño ejecutable para convertir videos MP4 a audios MP3.

## Requisitos
- Python 3.7 o superior
- pip

## Instalación

### Opción 1: Ejecutar el script directamente
```bash
pip install moviepy
python convertidor.py
```

### Opción 2: Crear ejecutable
```bash
# En Windows, ejecutar:
crear_ejecutable.bat
```

El ejecutable se creará en la carpeta `dist/`

## Uso
1. Ejecutar `convertidor.exe` (o `python convertidor.py`)
2. Hacer clic en "Seleccionar archivo"
3. Elegir el archivo MP4 a convertir
4. Hacer clic en "Convertir a MP3"
5. El archivo MP3 se guardará en la misma ubicación del original
